# 🎁 Giveaway Bot — Бот для розыгрышей

Telegram бот для проведения розыгрышей с реферальной системой и админ-панелью.

## ✨ Возможности

- 🎁 Создание розыгрышей с описанием, призом и датой окончания
- ✅ Участие пользователей в розыгрышах
- 🔗 Реферальные ссылки — каждый реферал = +1 билет к шансам
- 📊 Статистика рефералов в реальном времени (таблица лидеров)
- 🏆 Взвешенный случайный розыгрыш (больше рефералов = больше шансов)
- 🔐 Админ-панель для управления всем

## 🚀 Установка и запуск

### 1. Клонируй / скопируй файлы

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой config.py

Открой `config.py` и заполни:

```python
BOT_TOKEN = "токен от @BotFather"

ADMIN_IDS = [
    123456789,  # твой Telegram ID (узнать у @userinfobot)
]
```

Или используй переменную окружения:
```bash
export BOT_TOKEN="твой_токен"
```

### 4. Запусти бота
```bash
python bot.py
```

## 🎯 Как работает система рефералов

1. Пользователь нажимает **«🔗 Поделиться»** — получает уникальную ссылку
2. Друг переходит по ссылке и запускает бота
3. Пригласившему начисляется **+1 билет** в пул розыгрыша
4. При розыгрыше победитель выбирается **случайно**, но пропорционально количеству билетов

**Пример:** Если у тебя 5 билетов из 20 — твой шанс на победу 25%.

## 🗂 Структура файлов

```
giveaway_bot/
├── bot.py          # Основной файл бота (handlers)
├── database.py     # Работа с SQLite базой данных
├── config.py       # Токен и список админов
├── requirements.txt
└── README.md
```

База данных `giveaway.db` создаётся автоматически при первом запуске.

## 🛠 Деплой на сервер (пример systemd)

```ini
[Unit]
Description=Giveaway Telegram Bot
After=network.target

[Service]
WorkingDirectory=/home/user/giveaway_bot
ExecStart=/usr/bin/python3 bot.py
Restart=always
Environment=BOT_TOKEN=твой_токен

[Install]
WantedBy=multi-user.target
```
